bool LineLeader::takeSnapshot ( )

Take a snapshot, this command looks at the line under the sensor and stores the width and position of the line in sensors memory. Subsequently, sensor will use these characteristics of line to track it. This command inverts the colors if it sees a white line on black background. (PID parameters are not affected).
